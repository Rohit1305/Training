Number Guessing Game

INSTRUCTIONS OF THE GAME:

  • Guess the number by moving the slider.
  • The color changes to yellow if the guessed number is smaller.
  • The color changes to red if the guessed number is greater.
  • The color changes to green if the guessed number is correct.
  • You have a maximum of 6 attempts allowed.
